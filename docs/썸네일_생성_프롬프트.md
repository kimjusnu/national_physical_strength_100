# 운동 영상 썸네일 이미지 생성 프롬프트

15장. 생성 후 `public/thumbnails/01.webp` ~ `15.webp`로 저장하면 자동 연결됩니다.

## 규격

| 항목 | 값 |
|---|---|
| 비율 · 크기 | 3:2 · **480×320px** (표시 96×64px의 5배, 고해상도 대응) |
| 포맷 | webp 우선 (png도 가능 — 파일명 확장자만 맞추면 됨) |
| 저장 위치 | `public/thumbnails/` |
| 파일명 | `01.webp` ~ `15.webp` (아래 번호와 일치) |

## 공통 스타일 (모든 프롬프트 앞에 붙이세요)

서비스 디자인 시스템과 어긋나면 카드가 따로 놀기 때문에, 아래 블록을 **15장 전부 동일하게** 사용해야 합니다.

```
Flat vector illustration, minimal geometric style, single human figure in simple
silhouette form. Clean white background (#FFFFFF). Figure rendered in muted
teal-slate (#34464B). One accent color only: mint green (#04C584) used for the
equipment, motion line, or floor marker. No gradients, no shadows, no textures,
no text or numbers anywhere in the image. Generous negative space, figure
centered and occupying about 55% of frame height. Calm and instructional tone,
not energetic or sporty-aggressive. Consistent line weight throughout.
3:2 aspect ratio.
```

**이유:** 서비스는 흰 캔버스 + 틸-슬레이트 + 민트 그린 단일 강조색 체계입니다. 사진이나 그래디언트가 들어가면 데이터 화면의 신뢰감이 깨집니다. 이미지 안 텍스트는 한글이 깨져 나오므로 금지입니다.

## 생성 후 연결 방법

파일 15장을 `public/thumbnails/`에 넣고 알려주시면 코드에 연결하겠습니다. 직접 하시려면 `data/mockVideos.ts`의 각 레코드에서 `IMG_FILE_URL: ""` 를 `IMG_FILE_URL: "/thumbnails/01.webp"` 형태로 순번대로 채우면 됩니다.

일부만 생성하셔도 됩니다. 이미지가 없거나 로드에 실패한 카드는 기존 카테고리 아이콘으로 자동 폴백하도록 `components/result/VideoThumbnail.tsx`에 처리되어 있습니다.

## 참고: 실 API 연동 시

동영상 API(15108846)가 연결되면 `IMG_FILE_URL`에 공단 CDN의 실제 영상 썸네일이 들어옵니다. 여기서 만드는 이미지는 그전까지 자리를 채우는 용도이며, 실 API로 전환하면 자동으로 교체됩니다.
