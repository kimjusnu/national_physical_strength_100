# Claude Code 개발 프롬프트 (키 포함 · 바로 붙여넣기용)

아래 전체를 복사해 IDE의 Claude Code에 붙여넣으세요.

---

국민체육진흥공단(KSPO) 공공데이터 활용 경진대회에 출품할 **반응형 웹앱**을 만든다. 아래 스펙대로 새 프로젝트를 스캐폴딩하고 1차 기능까지 구현해줘.

## 서비스 개요
- 서비스명(가칭): **체력나이(Fitness Age)**
- 개념: 사용자의 체력을 입력/간이측정 받아, 국민체력100 공공데이터 분포와 비교해 **"당신의 체력 나이는 OO세"** 를 산출하고, 약점 항목에 맞는 운동처방 영상과 가까운 체력인증센터를 안내한다.
- 타깃: 전 연령 일반인 (모바일 우선, 큰 글씨 토글 등 접근성 고려)
- 심사 최적화 목표: 독창성 25%, 실용성 20%, 데이터활용 적절성 20%, 발전가능성 20%, 기술성 15%. → 국민체력100 데이터를 서비스 핵심 로직에 사용하고, 결과 공유 기능으로 바이럴/실용성을 살리고, 단일 플로우를 높은 완성도로 구현한다.

## 기술 스택
- Next.js (App Router) + TypeScript + Tailwind CSS
- 차트: recharts
- 배포: Vercel 기준. 환경변수로 API 키 관리.

## 환경변수 설정 (중요)
- 프로젝트 루트에 `.env.local` 생성하고 아래 값을 넣어라:
```
DATA_GO_KR_SERVICE_KEY=(발급받은 공공데이터포털 디코딩 키 — .env.local에만 보관, 문서/코드에 기재 금지)
USE_MOCK=true
```
- `.gitignore`에 `.env.local`이 포함돼 있는지 반드시 확인하고, 없으면 추가하라. 인증키는 절대 코드에 하드코딩하지 말고 서버사이드에서만 사용한다(공공데이터포털 API는 Route Handler를 통해 프록시 호출).
- `USE_MOCK=true`인 동안은 목업 데이터로 동작하고, `false`로 바꾸면 실제 공공데이터포털 API를 호출하도록 데이터 계층을 추상화하라.

## 공공데이터포털 오픈API (실연동 대상)
서버사이드(Route Handler `app/api/...`)에서 `DATA_GO_KR_SERVICE_KEY`로 호출. 키는 디코딩키 기준이며 요청 시 인코딩 처리에 유의.
- 국민체력100 체력인증센터 측정결과 정보: data.go.kr/data/**15108938**/openapi.do (신장·체중·체지방률·허리둘레·혈압·악력·윗몸말아올리기·반복점프 등 + 연령/성별)
- 국민체력100 체력인증센터 측정건수 정보: **15114286** (지역별 센터/측정건수)
- 국민체력100 동영상 정보: **15108846** (운동처방/생애주기별 표준운동 영상)
개발 초기엔 실제 API 응답 스키마를 반영한 **목업 데이터**로 전체 플로우를 먼저 완성하고, 이후 실 API로 교체한다.

## 사용자 플로우 (위저드)
1. **랜딩**: 서비스 소개 + "내 체력나이 측정하기" CTA. 데이터 출처(KSPO 국민체력100) 명시.
2. **기본정보 입력**: 성별, 실제 나이, 키, 몸무게.
3. **체력 항목 입력**: 악력(kg), 윗몸말아올리기(회/분), 반복옆뛰기 또는 제자리멀리뛰기, 체지방률(%) 또는 허리둘레(cm), (선택) 왕복오래달리기 단계. 각 항목에 "측정 방법 안내" 모달 제공.
4. **결과 화면**:
   - 큰 숫자로 **체력나이** (실제나이 대비 +/- 강조, 색상 피드백)
   - 항목별 백분위 레이더/바 차트(recharts)
   - 강점/약점 요약 문구
   - **맞춤 운동처방 영상 3~5개** (동영상 API, 약점 항목 기준 추천)
   - **가까운 체력인증센터** 안내(측정건수/센터 데이터, 지역 선택 또는 위치 기반)
   - **결과 카드 이미지 저장/공유** 버튼(SNS 바이럴용)

## 체력나이 산출 로직
- 국민체력100 측정결과로 **성별×연령대×항목별 백분위 기준테이블** 구성.
- 사용자 항목 값을 자신의 성별 분포에서 백분위 환산.
- 각 항목에서 "그 성적이 평균이 되는 연령대"를 역산 → 항목별 체력나이.
- 항목별 체력나이의 가중평균으로 **종합 체력나이** 산출(가중치는 상수 분리).
- 로직은 `lib/fitnessAge.ts` 순수함수로, 기준테이블은 `data/normTables.ts`로 분리.

## 이번 단계에서 구현할 범위 (1차)
프로젝트 스캐폴딩 + `.env.local`/`.gitignore` 설정 + 입력 위저드 + 체력나이 산출 로직 + 결과 화면(차트 포함)까지 **목업 데이터로 끝까지 동작**하게 구현한다. 운동처방 영상/센터/공유카드/실API 연동은 다음 단계에서 이어간다.
